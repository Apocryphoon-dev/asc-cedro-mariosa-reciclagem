# Imagens do Site ASC Ambiental

## Estrutura de pastas:
- `/images/hero/` - Imagens de banner/hero sections
- `/images/services/` - Imagens de serviços
- `/images/about/` - Imagens institucionais

## Fontes das imagens (temporárias):
Todas as imagens estão hospedadas no Unsplash.com (gratuitas para uso comercial)

## Para substituir por imagens próprias:
1. Salve suas imagens nesta pasta
2. Mantenha os mesmos nomes ou atualize o CSS
3. Recomendado: 1920x600px para hero banners
